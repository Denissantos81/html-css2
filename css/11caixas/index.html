Tipos de caixas

Temos as caixas do tipo:

box-level→Toda vez que se insere este tipo, ela vai para a proxima linha e oucupa o tamanho inteiro da tela.

inline-level→Quando é inserido este tipo de caixa ela não quebra a linha e só o espaço que foi destinado a caixa e o conteudo continua logo após a caixa
